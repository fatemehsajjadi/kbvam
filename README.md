
# kbvam.com

Online loan application website

[KbVam WebSite Deployed](https://kbvam.com)


## Deployment

To deploy this project run

```bash
  composer install
  php artisan key:generate
  php artisan migrate
  php artisan serve
  enjoy it!!!
```
